# RaspberryPi-OS

## Description

This is an toy Operating System for the Raspberry PI. 
It is based on [LumaOS](https://github.com/lucamienert/lumaos), my 32-bit OS.

## Usage

### Prerequisites

- Raspberry PI (or emulator)
- aarch64-linux-gnu-gcc

### Using

```bash
make run
```

### Cleaning the project

```bash
make clean
```
